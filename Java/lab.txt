Question # 3

With recursion
Program factorial
1.  Assign that number for the factorial to variable num
2.  If num == 0, return 1
3.  Otherwise result = num times factorial(num - 1)
4.  Return result

Without recursion
1. Assign the number for the factorial to variable num
2. Assign number 1 to variable result
3. Begin loop (i = 1; i <= num; num ++)
4. result = result times i
5. End loop
6. Print result

Example Java code:

public class Factorial {
    public static void main(String[] args) {
        System.out.println(calc1(5));
        System.out.println(calc2(5));
    }
    public static int calc1(int num) {
        if (num == 0)
            return 1;
        int result = num * calc1(num - 1);
        return result;

    }
    public static int calc2(int num) {
        int result = 1;
        for (int i = 1; i <= num; i++) {
            result *= i;
        }
        return result;

    }
}
